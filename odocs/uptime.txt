One Day One Command 
===================

uptime -- Show how long the system has been running.

Summary:

uptime gives a one line display, like this:

18:16:38 up 28 days, 8:18, 6 users, load average: 0.42, 0.21, 0.15

The Current Time, How long the system has been running, How many 
users are currently logged on, and the system load averages for 
the past 1, 5, and 15 minutes.

Examples:

$ uptime -- Show the uptime details.

Read: man uptime
